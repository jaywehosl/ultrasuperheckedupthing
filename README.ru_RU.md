# Ultra Uber Panel (3X-UI Antigravity Edition)

[English](/README.md) | [Русский](/README.ru_RU.md)

Продвинутая веб-панель управления серверами [Xray-core](https://github.com/XTLS/Xray-core) в премиальном стиле визуальной темы **Google Antigravity**, с одностраничным интерфейсом (SPA) и исправленной локализацией телеметрии.

Это специальная редакция панели 3X-UI. Мы ушли от разделения интерфейса на множество отдельных страниц в пользу единой плавной ленты, добавили адаптивный стеклянный дизайн (Glassmorphic) и интегрировали физический движок частиц с гравитационным центром.

---

## Ключевые изменения

* **Гравитационный движок частиц**: Глобальный элемент `<GravityVortexCanvas />` визуализирует плавное вращение колец частиц вокруг центра экрана. Траектория динамически искажается волнами синуса и закручивается в локальные вихри вокруг курсора мыши.
* **SPA-архитектура (Единая лента)**: Разделы телеметрии, входящих подключений (inbounds), клиентов, групп, узлов и системной статистики объединены в одну общую вертикальную страницу.
* **Премиальный эффект стекла**: Полупрозрачные карточки с размытием заднего фона (`backdrop-filter: blur(16px)`) органично накладываются поверх анимации частиц в темной, светлой и ультра-темах.
* **Исправления локализации**: Полностью переведены заголовки системной телеметрии, статистики и диагностики на русский и английский языки (устранены сырые системные ключи в интерфейсе).
* **Скрипты автоматизации**: Добавлены локальный запуск (`run.sh`) для разработчиков и скрипт компиляции/установки службы на сервере (`deploy.sh`).

## Быстрый старт

### 1. Установка в одно действие (Рекомендуется)
Вы можете развернуть готовую сборку Antigravity-издания на любом сервере Linux AMD64 одной командой:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/jaywehosl/ultrasuperheckedupthing/main/install.sh)
```

Инсталляционный скрипт автоматически скачает наш кастомный архив релиза из GitHub Releases, настроит систему инициализации systemd и запустит службу панели.

### 2. Запуск локально (Режим разработки)
Требуются установленные Go 1.21+ и Node.js 18+.

```bash
# Клонирование репозитория
git clone https://github.com/jaywehosl/ultrasuperheckedupthing.git
cd ultrasuperheckedupthing

# Сборка интерфейса и запуск локального dev-сервера
chmod +x run.sh
./run.sh
```

### 3. Сборка и деплой из исходников на сервере
Для сборки фронтенда, компиляции бинарного файла Go и регистрации службы systemd на сервере:

```bash
sudo chmod +x deploy.sh
sudo ./deploy.sh
```

## Структура проекта

* `/frontend`: Одностраничное приложение (SPA) на React + TypeScript.
* `/web`: Веб-сервер Go, REST API, контроллеры и файлы локализации.
* `/xray`: Модуль интеграции с процессом Xray-core.
* `/util`, `/logger`, `/sub`: Логирование, вспомогательные утилиты и подписки.
* `/install.sh`, `/update.sh`: Стандартные инсталляционные скрипты.
